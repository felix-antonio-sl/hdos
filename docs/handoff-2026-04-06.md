## Decisiones Arquitecturales

1. **PG es la fuente canónica** — correcciones directo en PG, no re-derivar desde CSVs
2. **Estrictos como base** — solo datos con RUT en `db/hdos.db` entran a PG
3. **Framework categorial** — cada fase es un Functor con PathEquations verificables
4. **Proveniencia como natural transformation** — cada dato trazable a su origen
5. **DDL fix** — 30 FKs forward-ref convertidas a ALTER TABLE (el DDL original no ejecutaba)

## Problemas Conocidos

1. **61 estadías solapadas** — rechazadas por `EXCLUDE USING gist`. Son reingresos legítimos. Pendiente: ajustar rangos `[)` en vez de `[]`, o relajar constraint
2. **8 fechas invertidas** — corregidas por swap en F₃ (fecha_egreso < fecha_ingreso)
3. **1 locality_id duplicado** — en CSV canonical (1660→1659)
4. **PE-F3-COUNT es warning** — 777 ≠ 838, no bloquea migración

## Para Continuar en el Servidor

### 1. Clonar/pull el repo

```bash
cd /ruta/al/repo
git pull origin main
```

### 2. Levantar PG (si no existe)

```bash
docker run --name hodom-pg \
  -e POSTGRES_DB=hodom \
  -e POSTGRES_USER=hodom \
  -e POSTGRES_PASSWORD=hodom \
  -p 5555:5432 \
  -d postgres:14-alpine

docker network connect web hodom-pg
```

### 3. Instalar psycopg en venv

```bash
python3 -m venv .venv
.venv/bin/pip install "psycopg[binary]>=3.1" streamlit pandas
```

### 4. Ejecutar migración

```bash
.venv/bin/python scripts/migrate_to_pg/run_migration.py \
  --db-url postgresql://hodom:hodom@localhost:5555/hodom
```

Esperado: 4 phases, ~4700 objects, ALL PASSED (PE-F3-COUNT es warning).

### 5. Aplicar inferencia de sexo post-migración

La inferencia ya está incorporada en F₂ via `input/manual/sexo_inferred.csv`. Se aplica automáticamente al ejecutar la migración.

### 6. Levantar dashboard

```bash
docker compose -f docker-compose.migra.yml up -d --build
```

O sin Docker:
```bash
DATABASE_URL=postgresql://hodom:hodom@localhost:5555/hodom \
  .venv/bin/streamlit run apps/streamlit_migration_explorer.py --server.port=8504
```

### 7. Verificar

```bash
.venv/bin/python -m pytest tests/test_migration/ -v  # 47 tests
```
