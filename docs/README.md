# Documentacion HODOM

## Organizacion

- `models/`: modelos conceptuales y estructuras de datos estabilizadas.
- `specs/`: referencias normativas, REM y SQL auxiliar vinculado al dominio de la base.

## Documentos principales

- `models/modelo-integrado-hodom.md`: descripcion principal del modelo integrado.
- `../db/hodom-integrado-pg-v4.sql`: dump PostgreSQL versionado conservado como artefacto principal de base.
- `models/erd-modelo-integrado-hodom.html`: diagrama ERD navegable del modelo integrado.
- `models/fhir-erd-hospitalizacion-domiciliaria.html`: vista ERD orientada a interoperabilidad.
- `specs/manual-rem-2026.md`: referencia operativa del REM.
- `specs/rem-a21-c1-estructura-datos.md`: estructura de datos REM relacionada.

## Regla practica

La documentacion de este directorio debe explicar la base actual o su contexto normativo directo. El material historico u operacional retirado ya no forma parte del alcance del repositorio.
