# HODOM

Repositorio centrado en la base de datos HODOM y su documentacion vigente.

## Alcance actual

- `db/`: artefactos versionados de la base PostgreSQL.
- `docs/`: modelo de datos, ERD, referencias normativas y estructura de reporting relacionada.

## Estado del repositorio

- No se mantienen pipelines de migracion, dashboards, scripts operativos ni suites de pruebas.
- El repositorio conserva solo el estado actual de la base y la documentacion necesaria para entenderla.
